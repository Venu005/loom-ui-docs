---
title: Next.js
description: Install and configure loom ui in your Next.js project
---

<Steps>

<Step>

## Create a new Next.js project

```bash
npx create-next-app@latest
```

</Step>

**_You need to have tailwindcss installed in your project to use loom ui._**

<Step>

## Initialize Loom UI

Use the init command to install the Loom ui components and configuration.

```bash
npx @venusai/loom-ui init
```

This will configure components.json and the file structure similar to the example given below below.

<Files>
  <Folder name="src" defaultOpen>
    <Folder name="app">
      <File name="layout.tsx" />
      <File name="page.tsx" />
      <File name="global.css" />
    </Folder>
    <Folder name="components" defaultOpen>
      <Folder name="loomui" defaultOpen>
        <File name="button.tsx" />
      </Folder>
    </Folder>
    <Folder name="lib" defaultOpen>
      <File name="utils.ts" />
    </Folder>
    <File name="components.json" />
  </Folder>
</Files>

</Step>

<Step>

## That's it!!

You can now use the Loom UI components in your project.

```bash
 npx @venusai/loom-ui add button
```

The command above will add the Button component to your project. You can then import it like this:

```tsx title="page.tsx"
import { Button } from "@/components/loomui/button"; // [!code highlight]
export default function Home() {
  return (
    <div>
      <Button>Click me</Button> {/* [!code highlight] */}
    </div>
  );
}
```

</Step>

</Steps>
